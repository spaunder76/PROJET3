En tant qu'étudiant sur Learn Home :

    L'utilisateur cible est un étudiant qui utilise la plateforme Learn Home pour ses études.

Je veux pouvoir discuter instantanément avec des bénévoles et d'autres élèves sur l'interface de chat :

    L'utilisateur veut avoir la possibilité de discuter avec d'autres personnes sur la plateforme via une interface de chat.
    Il/elle veut que cette fonctionnalité soit disponible en temps réel pour pouvoir poser des questions et obtenir de l'aide rapidement pour ses devoirs.

Je veux également être capable d'accéder à l'historique de mes conversations :

    L'utilisateur veut pouvoir accéder à ses conversations passées avec d'autres personnes sur la plateforme.
    Il/elle veut pouvoir consulter l'historique de ses conversations pour retrouver des informations importantes.

Je veux pouvoir ajouter ou supprimer des contacts :

    L'utilisateur veut pouvoir ajouter des personnes à sa liste de contacts pour faciliter la communication.
    Il/elle veut également pouvoir supprimer des contacts s'il/elle ne souhaite plus communiquer avec eux.

En outre, je veux que chaque message sur l'interface de chat contienne une photo de profil de l'expéditeur à côté du message :

    L'utilisateur veut que chaque message envoyé sur la plateforme via l'interface de chat soit accompagné d'une photo de profil de l'expéditeur à côté du message.
    Cette fonctionnalité aidera l'utilisateur à identifier l'auteur de chaque message plus facilement.

Je veux également que chaque message indique s'il a été lu ou non :

    L'utilisateur veut savoir si ses messages ont été vus par les destinataires.
    Il/elle veut que chaque message envoyé via l'interface de chat indique s'il a été lu ou non.

Enfin, je veux que chaque message sur l'interface de chat inclue l'horodatage :

    L'utilisateur veut pouvoir connaître l'heure d'envoi de chaque message via l'interface de chat.
    Il/elle veut que chaque message soit horodaté pour avoir une idée de la chronologie des conversations.
